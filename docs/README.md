# SnipSnap Website

Product page for SnipSnap, built with plain HTML/CSS/JS.

## Local Development

Just open `index.html` in your browser. No build step needed.

## Deployment

The site auto-deploys to GitHub Pages via `.github/workflows/pages.yml` whenever changes are pushed to the `docs/` directory on the master branch.

Live at: https://portablesheep.github.io/snipsnap

## Design Notes

- Pure HTML/CSS/JS (no frameworks)
- Apple-inspired design system
- Responsive and accessible
- Fast and lightweight (~20KB total)

## Avoiding AI Tells

Copy written to avoid common AI-generated phrases:
- ❌ "seamless", "leverage", "revolutionize", "game-changer"
- ✅ Casual, conversational tone
- ✅ Specific examples and real frustrations
- ✅ Personality and humor
